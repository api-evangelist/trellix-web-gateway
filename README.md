# Trellix Web Gateway (trellix-web-gateway)
Trellix Web Gateway (formerly McAfee Web Gateway) provides advanced threat  protection and secure web access for enterprises. It offers URL filtering,  malware detection, data loss prevention, and cloud security capabilities  through a comprehensive web security platform.

**URL:** [Visit APIs.json URL](https://www.trellix.com/products/web-gateway/)
